# macOS-PasswordManager
Password Manager for macOS

### Features :
- [x] Touch ID / Password Sign in
- [x] Save Passwords & visibile state in Core data
- [x] Search 
- [x] Edit 
- [x] Delete
- [x] Press on cell when password is visible to copy
